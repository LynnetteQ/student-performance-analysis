Student Performance Analysis
Objective:
Analyze student data to identify trends in academic performance and key influencing factors.
Tools Used:
Microsoft Excel
Pivot Tables
Data Visualization
Key Questions:
Does test preparation impact scores?
How does lunch type affect performance?
Are there differences by gender?
Key Findings:
Students who completed test prep scored higher across all subjects
Students with standard lunch performed better than free/reduced lunch
Math scores were generally lower than reading and writing
Recommendations:
Increase access to test prep programs
Provide additional math support
Address resource gaps impacting student performance
